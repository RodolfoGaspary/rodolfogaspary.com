Sobre mí

Soy Rodolfo Gaspary. Desarrollador web independiente en Lima.

Como creador de páginas web perú, construyo sitios a medida pensados para que los negocios capten más clientes mediante un diseño de sitios web rápidos. Trabajo de forma selectiva y honesta, con pocos proyectos a la vez, priorizando la calidad sobre el volumen.

"Una web no es un proyecto de diseño: es una herramienta de negocio. Por eso mi trabajo empieza siempre con una pregunta antes de escribir una línea de código — ¿qué quieres que esta web logre para ti?"
